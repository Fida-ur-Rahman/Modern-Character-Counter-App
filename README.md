# Modern Character Counter App

A modern and responsive **Character Counter Web Application** built using **HTML, CSS, and JavaScript**. This app provides real-time character and word counting with a clean user interface and visual limit indicators.

##  Features

-  Live character count  
-  Real-time word count  
-  Progress bar showing character usage  
-  Visual warning when nearing or exceeding the limit  
-  Fully responsive design for mobile, tablet, and desktop  
-  Modern glass-style user interface  

##  Technologies Used

- **HTML5** – Structure of the application  
- **CSS3** – Modern styling and responsive layout  
- **JavaScript (Vanilla JS)** – Real-time text analysis and UI updates
